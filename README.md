# UK-Train-Analytics-Dashboard
This project provides a comprehensive analysis of UK train performance and revenue. Using Power BI, I transformed raw data into an interactive dashboard to help stakeholders understand travel patterns, payment preferences, and station performance.
##🚀 Project Workflow
Data Loading & Cleaning: Imported the dataset, handled missing values, and standardized data types for accuracy.

Data Understanding: Analyzed journey status, ticket types, and pricing to determine the most impactful business drivers.

DAX & KPIs: Developed custom measures using DAX to track:

Total Revenue: £742K

Rolling 3M Revenue: £536K

On-Time Performance: 87%

MoM Growth: 34%

Design & Visualization: Managed shapes, themes, and layout to create a professional, high-contrast dashboard for better readability.

📊 Key Insights
Station Performance: London Kings Cross is the top-performing station by revenue.

Payment Trends: Credit Cards remain the most popular payment method at roughly 63%.

Reliability: The majority of journeys (76%) are on time, with "Delayed" and "Cancelled" statuses clearly visualized for further investigation.
